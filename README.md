## 프로젝트 내용
 - 간단한 CUI(Console User Interface)기반 계산기
 - 지원하는 기능은 더하기, 빼기, 곱하기이며, 나누기 기능은 지원하지 않음
 - 정수만 입력되며, 한 연산에서는 한 종류의 연산자만 들어옴
 - ‘=’을 입력받으면 결과를 출력함
 - 연산 기능과 무관한 이스터에그가 존재함
